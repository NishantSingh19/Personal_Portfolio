About -

You can also get personalized movie suggestions with posters using the OMDb API. The project combines data analysis and visualization to make exploring Netflix’s content simple, fun, and informative.

About -

This interactive dashboard analyzes global video game sales data using Streamlit, Pandas, and Plotly. It helps explore insights about the top-selling games, platforms, genres, and publishers across different regions. Users can switch between tabs to see trends, compare regional sales, and discover which genres or platforms dominated each year. The dashboard turns raw data into clear, visual stories that make exploring the gaming industry both fun and insightful.
